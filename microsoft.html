Основні дати в історіі компаніі

1975

Друзі по навчанню Біл Гейтс та Пол Ален заснували компанію Microsoft. Першим їх продуктом був інтерпритатор мови Basic.

1980

Microsoft підрисала договір з IBM на розробку операційноі системи MS-DOS.

1983

Реліз першої версіі MS Word.

1985

Реліз першої версіі Excel. Версія була призначена тільки для компьютерів Macintosh.

1995

Реліз Windows 95 з оновленим інтерфейсом за кнопкою "Пуск".

1995

Біл Гейтс став найбагатшою людиною у світі та зберігав цей статус до 2017 року.
